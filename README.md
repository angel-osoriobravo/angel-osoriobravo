## Home Lab (The Farm)
I created a homelab four years ago that has been evolving ever since. It started with an old Dell desktop with a Celeron chip, and now I am running two separate servers, each with its own purpose. To make it easier for me to remember what each one is for, I named the servers after my pets, which is why the entire environment is titled 'The Farm.' There have been different iterations of each setup over time, which is why they also have a version number attached to them.

### My Servers
*   **(Pollo_Server2 )Dell OptiPlex 7060:** This is my main hypervisor server, and it runs Proxmox.
*   **(Nike_Server) Custom PC:** This is my main storage and docker server, which runs UNRAID with 10TB of storage.
